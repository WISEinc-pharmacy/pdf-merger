PDFツール

このツールでできること
  • PDF結合 — 複数のPDFを1つにまとめます。ドラッグで順番を変更できます
  • PDF分割 — ページ範囲を指定して、必要な部分だけ抽出します
  • 画像→PDF — JPEG・PNG画像をPDFに変換します。複数画像を1つのPDFにまとめることも可能
  • PDF→画像 — PDFの各ページをJPEGまたはPNG画像に変換します。解像度（dpi）を選択できます
  • Office→PDF — Word・Excelファイルを簡易的にPDFへ変換します（複雑なレイアウトは印刷経由が確実）
  • フォント変換 — テキストを全ページ画像化し、楽楽明細の郵送代行で文字化けしないPDFに変換します
  • ベクター化（.ai/.svg） — JPG・PNG画像（ロゴ・線画）をベクター形式に変換します。Illustratorで開ける.aiファイルをブラウザ内で生成します
  • 送付先付番 — 送付先フォルダ内の全PDFに、送付先ごとの番号を自動で付けます（楽楽明細の配信先判定用）。送付先マスタ表はツール内で入力・保存できます
  • AI要約 — PDFの内容をAIが読み取り、要点をまとめて表示します（近日追加予定）

※ すべてブラウザ内で処理されます。ファイルが外部に送信されることはありません（AI要約を除く）。

使い方：報告書PDFをドラッグ&ドロップ → 「変換開始」→ ダウンロード。
フォントが全て画像化されるため、楽楽明細の郵送代行で文字化けしません。複数ファイル一括OK。
楽楽明細 郵送代行 フォント判定基準を確認する

郵送代行 OK のケース

  • ✓ フォント情報なし（文書のプロパティ→フォントタブが空欄）
  • ✓ 埋め込みサブセット or 埋め込み、かつエンコーディングが「ビルトイン以外」

郵送代行 NG → このツールで変換

  • ✗ エンコーディングが「ビルトイン」
  • ✗ 「埋め込みサブセット or 埋め込み」の記載がない

※ Acrobat Reader の「ファイル→プロパティ→フォント」で確認できます。

使い方：JPG・PNG画像をドロップ → オプション設定 → 「ベクター化開始」→ .ai / .svg をダウンロード。
ロゴ・線画・単色イラスト向け。写真は不向きです。Illustratorでそのままベクターとして開けます。
⚠ フォント・レイアウトが崩れる場合があります
複雑なレイアウトのファイルは、PCの「印刷」→「PDFとして保存」でPDF化する方がより正確です。
簡易的な変換としてご利用ください。

※ 大きな画像は変換に数秒かかります。ロゴ・線画（シンプルな絵）向け。写真は向きません。

使い方：①下の「送付先マスタ」に 送付先名（フォルダ名に含まれる語）と番号 を入力 → ②送付先フォルダをドロップ → ③「付番してZIP出力」。
フォルダ内の全PDFのファイル名に、その送付先の番号が自動で付きます。マスタと処理はすべてこのPC内で完結し、外部やネットには一切送信されません。
送付先マスタ（送付先名 → 番号の対応表） 自動保存（このブラウザに保存されます）
送付先名（フォルダ名に含まれる語） 番号
ファイル名の付け方：

使える語：{name}=送付先名（マスタの名前） / {folder}=フォルダ名そのまま / {code}=番号 / {original}=元のファイル名
例）{name}_{code}_{original} → つなぐ薬局柏_1_1234_訪問日.pdf
